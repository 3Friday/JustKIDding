The browser is the program you use to see the internet. You click its icons each time you want to open a page. Now we will show you some cool stuff that it can help you do.

Choose your browser:

  • Internet explorer

    Click on the keyboard button F12 or go to the menu and click F12 Developer Tools

  • Chrome

    Click on the keyboard button F12 or go to the menu icon with the 3 lines on the top right, then select Tools>Programmer tools

  • Mozilla

    Click on the keyboard button F12 or go to the menu icon with the 3 lines on the top right, then select Developper>Inspector
