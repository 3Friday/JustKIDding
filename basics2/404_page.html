HTTP Status Codes:

  • 200- 299 Success
  • 300- 399 Redirect
  • 400- 499 User error
  • 500 Server

So the 404 Not Found error is displayed when a certain page is not found. If all goes well, this page is never see. So it is an opportunity to be creative.
