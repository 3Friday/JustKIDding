Ruby on Rails

As the name suggests this is a framework based on Ruby.

Created in 2004 by David Heinemeier Hansson

Convention over configuration: strict structure of naming

Django

Python's most popular framework.

Created in 2003 by Adrian Holovaty and Simon Willison, web programmers at the Lawrence Journal-World newspaper

How to create your app

rails new awesomeproject
startproject awesomeproject

How to run your server

bin/rails server
python manage.py runserver

Both can be run on multiple types of servers (RoR has its own called Webrick) and databases.

Both used the MVC design pattern. Model-View-Controller

Both use the DRY principle. Do Not Repeat Yourself.
