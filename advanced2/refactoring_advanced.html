What is refactoring?

Refactoring is rewriting your code to make it clearer and easier to modify.

When writing code we usually add another class or method one by one. But at some point, it all just becomes too complex. Then we want to review our code and make it easy to understand and by extention easy to build upon and quickly in case you want to add some feature some time in the future.
