Security

If we are making a more complex app, we might want to protect some data.

For example, we might want no one to see a user's password, name or any other sensitive information.

We also want our site to be accessible. But if the topic discussed is controversial, someone can decide to take it down.

We want to prevent that.

The MVC paradigm

Used frequently for web applications

Model (logic, business rules): describes the associations between the objects and manipulates the data

View (presentation): displayed to the user

Controller: manages the requests
