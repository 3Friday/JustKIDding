Find an open source project to work on

Now that you have sufficient experience with writing your own code, you might want to put it to the benefit of the community.

A good place to start looking is github's trending page where some of the most popular repositories are listed. And you can be sure they are active too.

You can also search by language or interest. Are you eager to go into game development? Maybe emojis are your thing? Or creating beautiful templates?

You don't have to do it all on your own. Join an existing project.

Another option is to join Free Code Camp where you can learn some more back end JavaScript and help out build websites for non profit organisations.

The choice is made. What's next?

First step is to read carefully the guidelines for contributing if any.

Most important, before doing anything, get to know the code. Fork the repository or clone it locally (on your computer).

Now, did you spot a bug? Do you have an idea about a new feature? Get digging!

Pull requests

You might receive comments before your code is accepted. Read them and make changes if necessary.
