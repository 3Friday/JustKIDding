Your Style

We already have arranged our elements but now the screen is too small.

@media all and (min-width: 800px) and (max-width: 1024px) {
...;
...;
...;
}

This tells the browser that if the window, also called viewport, is bigger than 800 pixels and smaller than 1024 pixels, to use the rules above.

Other case is when we are on a device that we can flip.

@media only screen and (orientation: portrait) {
...;
...;
...;
}
