Terminal, shell, command line: what is it and how to use it

Once upon a time if you wanted the computer to do something you had to type a command. Now we can just click. But typing still can give us power.

Folders in the terminal are called directories. You can:

  • print the working directory which means to see where you are at the moment: pwd
  • change the directory where you are: cd
  • see ,or list, the files that are inside the directory: ls
  • create (or make) and delete (or remove) the directory: mkdir and rmdir

Another useful thing is to manipulate files:

  • create them: touch
  • copy them: cp
  • move or rename them: mv
  • delete them: rm
  • cat
  • grep
  • view them: less and more

Wildcards are symbols we use to replace a character we don't know when searching: * and ?
