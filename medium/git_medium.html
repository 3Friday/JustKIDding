When you write a lot of code, eventually you will want to go back to a previous version that you wrote. For that, we use a little program called git.

To start working on a project you can:

  • create a folder with some documents in your computer and write in the terminal "git init" (without the quotes;
  • go to github and create a new repository and add there some files and download them with "git clone" and the link on the right side of the project.

After you make some more changes you should add them to the staging area.
