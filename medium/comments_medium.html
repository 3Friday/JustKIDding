justKIDding

People forget. People use other's code. That is why we want to make notes in our code. They should not appear on your page but are helpful. We call them comments.

    This is how you write a comment in:

  • HTML:

    <!-- Here is a comment in HTML. You can use it to mark the beginning and end of an element, put a reminder on what to do, and so on. -->

  • CSS:

    /* This is a CSS comment that can be written on a single or multiple lines. You can use it to separate the section for the main page and other pages. It also makes your code more readble. */

  • Javascript:

    // Using these two slashes, you can make a comment on a single line, else you can use the same syntax as in the css file.

You do not have to repeat the obvious, but commenting your code can be useful for you at a later time or to someone else that want to continue or expand the project you are working on.

So leave tips! Leave whitespace. And remove the confusion!
