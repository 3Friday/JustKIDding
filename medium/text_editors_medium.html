The text editor is usually the place where you should write your code. Using a plain and simple text editor such as Notepad is fine. But using a more fancy one can automate some of the things you do, which means that it helps you write easier and faster. It gives you hints. Text editors are also colo(u)r coded which means that for example the elements are blue, the plain text is white, etc.

Some of the most popular are: Sublime text, Notepad++, Atom (which is github's version, we will talk about github later) Vim and Emacs.
